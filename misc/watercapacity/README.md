## Problem

Given an array, print how many units of water it can hold. array representing hight of walls

1 0 1 can hold 1 unit of water
3 1 0 2 can hold 3 units of water
3 2 1 cannot hold any water so the answer is 0



Sample test cases:

5 1 2 1 3 3 4 --> 10
7 4 3 3 2 1 1 --> 0
3 7 3 1 4 8 2 --> 13
9 2 5 2 1 4 8 --> 26
